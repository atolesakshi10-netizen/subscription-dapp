# Decentralized Subscription Payment System

A blockchain-based subscription payment system where users can subscribe, renew, and cancel plans without intermediaries.

## Features
- Connect wallet using MetaMask
- Load contract details
- Subscribe to a plan
- Renew subscription
- Cancel subscription
- Check subscription status

## Tech Stack
- Solidity
- Node.js
- Express.js
- HTML, CSS, JavaScript
- Ethers.js
- MetaMask
- Sepolia Testnet

## Project Structure
subscription-dapp/
├── contract/
├── frontend/
├── server.js
├── package.json

## How to Run
1. Install dependencies:
   npm install

2. Start backend:
   node server.js

3. Open frontend in browser.

## Note
Do not upload private keys, .env files, or node_modules.
